# README

ArchiMig lets you model a sequence of steps with which you would like to migrate an existing system architecture into a new one. Each steps describes which parts of your architecture should be added, moved, or deleted. As a result, ArchiMig will create diagrams, one for each modeled step.

To get a first impression of how this could look like, download the repo, open it in your favorite Maven-supporting IDE and run the CaaMigrator class. It will (hopefully) output some AsciiArt like this:
